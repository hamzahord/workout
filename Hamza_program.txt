Mon programme est basé sur trois séances par semaine.
Première séance:
Grands muscles : épaules
exercices : developpés militaire:8 rep, levés sur les cotés + vers l'avant 5rep chacun, facepull 10 rep, marche du fermier + levé d'épaules
Petits muscles : triceps (entier), biceps(longue portion), avant bras.
exercices : exos long portion + vaste interne avec les cables, exos vaste externe en alternance.
marteau droit + sur les cotés, curle inversés
Deuxième séance : 
Grands muscles : Jambes(tout)
exerices : squatt, soulevé de terre, presse horizontale, leg extension
Petits muscles : triceps, biceps, rappel avant bras.
exercices : exos long portion + vaste interne avec la barre,  exos vaste externe.
marteau droit + sur les cotés en alternance, curle 
Troisième séance :
Grands muscles : pecs et dos
exercices : developpé couché à la barre + tractions, developpé incliné, machine
